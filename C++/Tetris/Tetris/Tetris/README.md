# Tetris

A simple console-based Tetris game written in C++.

## Description

This project is a classic implementation of the Tetris game that runs in the Windows console. It features a scoring system, increasing difficulty, and a preview of the next piece.

## How to Compile and Run

1.  Open the `Tetris.sln` file in Visual Studio.
2.  Build the solution (F7 or Build > Build Solution).
3.  Run the application (F5 or Debug > Start Debugging).

## Controls

-   **A** or **Left Arrow**: Move the piece left
-   **D** or **Right Arrow**: Move the piece right
-   **S** or **Down Arrow**: Move the piece down faster
-   **W** or **Up Arrow**: Rotate the piece
-   **P**: Pause the game
-   **Esc**: Exit the game
